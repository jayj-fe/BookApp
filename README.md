## Book App

React.js로 만든 도서 검색 앱 프로젝트입니다.<br>
React.js로 개발하고 Naver Api를 이용하여 검색기능을 구현하였습니다.

<br />

## Technology Stack

- React.js, Axios, HTML, SCSS, netlify

<br />
